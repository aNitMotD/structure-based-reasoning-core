# Structure-Based Reasoning Core

## English

This repository collects design artifacts
produced through a structure-based reasoning process.

It does **not** contain:
- the full reasoning engine itself
- a complete methodological framework
- prescriptive instructions
- executable or operational systems

The materials here represent **designed outputs only**,
including boundary declarations, constraint specifications,
and documented edge cases.

Directory grouping reflects document type, not chronological order.

---

## Scope and Positioning

This repository is:
- **Non-Prescriptive** — it does not instruct how to think, decide, or act
- **Non-Executable / Non-Operational** — it is not a system, tool, or workflow
- **Non-Authoritative** — it does not act as a proxy for human judgment

No part of this repository
should be treated as a complete, standalone, or self-sufficient unit.

---

All materials in this repository are derived outputs.

always remains with the reader or user.

Attribution and ownership follow a conditional model:

- Transfer of ownership requires explicit responsibility acceptance

# Structure-Based Reasoning Core

## 한국어

이 레포지토리는
구조 기반 사고 과정을 통해 생성된
설계 산출물을 정리한 저장소이다.

본 레포지토리는 다음을 포함하지 않는다:
- 사고엔진 자체
- 완결된 방법론 프레임워크
- 사고·판단·행동을 지시하는 내용
- 실행 가능하거나 운영 목적의 시스템

여기에 포함된 자료는
**설계 산출물만을 기록**하며,
경계 선언, 제약 명세,
엣지 케이스 문서로 구성된다.

디렉토리 구분은 형성 순서가 아니라 기록의 성격을 기준으로 한다.

---

## 범위 및 위치 정의

본 레포지토리는 다음 특성을 가진다:

- **비지침적(Non-Prescriptive)** — 사고나 판단을 지시하지 않는다
- **비실행·비운영(Non-Executable / Non-Operational)** — 시스템이나 도구가 아니다
- **비권위적(Non-Authoritative)** — 인간 판단을 대리하지 않는다

본 레포지토리의 어떠한 일부도
완결된 단위이거나
독립적으로 충분한 내용으로
취급되어서는 안 된다.

---

## 책임 및 파생 관계

본 레포지토리에 포함된 모든 자료는
사고 결과로서의 파생 산출물이다.

해석, 판단, 적용에 대한 책임은
항상 독자 또는 사용자에게 귀속된다.

귀속 및 소유는 조건부 모델을 따른다:
- 조직 외부에서 생성된 산출물은 개인 귀속
- 조직 내부에서 생성된 산출물은 조직 귀속
- 소유 이전은 명시적인 책임 수용을 전제로 한다

세부 내용은 관련 선언 문서를 참고한다.
