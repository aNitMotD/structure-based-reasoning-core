# structure-based-reasoning-core

본 레포지토리는
구조 기반 사고 활동에서 도출된
선별된 설계 산출물을 포함한다.

작성자의 사고엔진 전체를 설명하거나,
방법론 또는 운영 프레임워크를
제공하는 레포지토리는 아니다.

---

## 해석 가드레일

본 레포지토리의 모든 내용은
다음 문서를 전제로 해석되어야 한다:

- INTERPRETATION-GUARDRAILS.md

해당 문서는
자료의 해석 방식과
해석해서는 안 되는 경계를 정의한다.
반드시 선행해서 확인할 것.

---

## 레포지토리 상태

본 레포지토리는 현재 정리 중이다.

현재는 전체 구조 정리의 일부로서
ZR 및 ZR-UX 관련 설계 산출물이
우선적으로 정리되어 있다.

향후 추가 자료, 부록,
또는 경계 고지가 포함될 수 있으며,
이는 기존 핵심 주장이나
해석 가드레일을 변경하지 않는다.
