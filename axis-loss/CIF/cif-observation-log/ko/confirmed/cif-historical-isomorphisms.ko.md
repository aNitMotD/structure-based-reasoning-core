# CIF Historical Isomorphisms — Observational Set

> 본 문서는 역사적 사건의 평가·교훈·도덕 판단을 포함하지 않는다.  
> 각 항목은 Constraint Inheritance Failure(CIF)와의 **구조적 동형성** 만을 기록한다.

---

## 1. Galileo — Heliocentrism Case

- **사건 식별자**: Galileo-Heliocentrism
- **초기 제약(Constraint Set)**:
  - 성서 해석 기반 우주론이 진리 판단의 상위 제약
  - 교회 권위 = 지식 정합성의 최종 판정자
- **새 정보/구조 입력**:
  - 망원경 관측 기반 지동설 모델
- **제약 상속 실패 지점**:
  - 관측 가능성·재현성 제약이 신학적 판단 체계로 상속되지 않음
- **사후 처리 경로**:
  - CIF-d
- **대체 프레임**:
  - 진위 판단 → 신성모독/질서 위반 프레임
- **결과**:
  - 이론 배제, 개인 처벌
- **메타 주석**:
  - 과학 vs 종교 문제가 아니라 제약 우선순위 고착 사례

## 2. Lavoisier — Revolutionary Tribunal Case

- **사건 식별자**: Lavoisier-Revolution
- **초기 제약(Constraint Set)**:
  - 혁명 정합성·정치적 순수성 제약
- **새 정보/구조 입력**:
  - 근대 화학 체계 확립, 공적 기여
- **제약 상속 실패 지점**:
  - 과학적 기여 제약이 정치적 판단 체계로 상속되지 않음
- **사후 처리 경로**:
  - CIF-d
- **대체 프레임**:
  - 기여도 → 구체제 인물/적대 계급 프레임
- **결과**:
  - 구조 배제, 사형
- **메타 주석**:
  - 개인 도덕성과 무관한 제약 충돌 고착

## 3. Semmelweis — Puerperal Fever Case

- **사건 식별자**: Semmelweis-Puerperal-Fever
- **초기 제약(Constraint Set)**:
  - 의학 권위·전통 이론 우선 제약
- **새 정보/구조 입력**:
  - 손 위생 도입 후 사망률 급감이라는 통계적 관측
- **제약 상속 실패 지점**:
  - 통계·경험적 제약이 권위 기반 판단 체계로 상속되지 않음
- **사후 처리 경로**:
  - CIF-d
- **대체 프레임**:
  - 데이터 → 모욕/비전문성/성격 문제 프레임
- **결과**:
  - 이론 배제, 개인 고립
- **메타 주석**:
  - 데이터 부재가 아니라 제약 수용 실패

## 4. Turing — Post-war Prosecution Case

- **사건 식별자**: Turing-Postwar
- **초기 제약(Constraint Set)**:
  - 법·도덕 규범 제약
- **새 정보/구조 입력**:
  - 전시 암호 해독 공헌, 국가 생존 기여
- **제약 상속 실패 지점**:
  - 공헌·기능적 가치 제약이 법적 판단 체계로 상속되지 않음
- **사후 처리 경로**:
  - CIF-d
- **대체 프레임**:
  - 공헌 → 규범 위반 프레임
- **결과**:
  - 처벌 및 사회적 제거
- **메타 주석**:
  - 기여와 규범을 병렬 처리하지 못한 사례

## 5. Wegener — Continental Drift Case

- **사건 식별자**: Wegener-Continental-Drift
- **초기 제약(Constraint Set)**:
  - 메커니즘 완비 요구 제약
- **새 정보/구조 입력**:
  - 지질·생물 분포의 관측적 정합성
- **제약 상속 실패 지점**:
  - 관측 정합성 제약이 설명 완전성 제약으로 상속되지 않음
- **사후 처리 경로**:
  - CIF-d
- **대체 프레임**:
  - 메커니즘 부재 → 전체 이론 무효 프레임
- **결과**:
  - 이론 장기 배제
- **메타 주석**:
  - 부분 제약 결손이 전체 구조를 소거한 사례
