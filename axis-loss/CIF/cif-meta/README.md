# CIF Meta

## Overview (EN)

This directory contains **meta-level records** of CIF.

It does not define CIF as a framework, methodology, or rule set.  
Instead, it records **structural perspectives** in which CIF appears
as a decision boundary, evaluation structure, or closure point.

Documents in this directory are:
- Not guidelines
- Not prescriptions
- Not responsibility attributions

They exist solely as **compressed structural observations**.

---

## 개요 (KO)

이 디렉토리는 CIF에 대한 **메타 레벨 기록** 을 담는다.

CIF를 프레임워크나 방법론으로 정의하지 않으며,  
CIF가 **결정 경계**, **판별 구조**, **닫힘 지점** 등으로
나타나는 관점들을 구조적으로 기록한다.

이 문서들은:
- 지침이 아니며
- 권고가 아니고
- 책임을 귀속하지 않는다

**CIF가 작동하는 구조적 관점에 대한 압축 기록** 일 뿐이다.
