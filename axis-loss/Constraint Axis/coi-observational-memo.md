# Observational Memo — Constraint Over-Inheritance

## Observation (EN)

A recurring pattern is observed where:
- Constraints appear in the output
- but are not grounded in the current constraint context

These constraints do not originate from
the locally defined constraint structure,
but appear as if inserted from outside the context.

This memo records a possible pattern:
constraint presence without contextual inheritance alignment.

(No definition, no classification, no prescription.)

## 관측 메모 (KO)

다음과 같은 패턴이 관측된다:

- 제약이 출력에 포함되지만,
- 현재 문맥의 제약 체계에 기반하지 않는 상태

이 제약은
국소적으로 정의된 제약 구조에서 유래하지 않으며,
문맥 외부에서 삽입된 것처럼 나타난다.

본 메모는 다음 가능성을 기록한다:
제약이 존재하지만,
문맥 상속 정합성이 없는 상태.
