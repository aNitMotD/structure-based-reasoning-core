# 관측 vs 예측

## 참고

본 문서는 관측과 예측의 구조적 차이를 설명하기 위한 개념 문서이다.

이 문서는 특정 대상, 발화, 문서, 판단을 분류하거나 판정하기 위한 운영 기준을 제공하지 않는다.

여기서의 구분은 설명을 위한 분석 축이며, 적용 규칙이나 평가 절차를 의미하지 않는다.

---

## 개요

본 문서는 관측과 예측의 차이를  
시간이 아닌 **구조 접근 가능성**, **상태 확정성**, **구조 폐쇄성** 의 관점에서 정리한다.

이 구분은  
SIF (Structural Inaccessibility Failure) 발생 시 나타나는  
인식 전이를 설명하기 위한 것이다.

여기서 중요한 것은  
미래를 다룬다고 해서 모두 예측이 되는 것은 아니라는 점이다.

어떤 미래 진술은  
구조가 이미 닫혀 있어서 관측의 연장으로 볼 수 있고,  
어떤 미래 진술은  
구조는 존재하더라도 시간 경로 추적과 상태 변화 판단이 필요하므로 예측에 속한다.

---

## 1. 핵심 구분

- 관측: 확정된 상태의 기술
- 예측: 비확정 상태에 대한 가정

이 차이는 시간이 아니라  
**상태의 확정성**, **구조 접근 가능성**, **구조만으로 결과가 닫히는가 여부** 에서 발생한다.

즉,

같은 미래에 대한 진술이라도  
어떤 것은 관측의 연장일 수 있고,  
어떤 것은 예측일 수 있다.

---

## 2. 관측

- 이미 발생했거나 확정된 상태를 기술
- 구조의 존재와 관계를 기록
- 시간은 과거 또는 현재에 고정
- 불확실성 없음
- 외부 가정이나 모델 개입 없음

→ “무엇이 있는가”

---

## 3. 구조 기반 미래

> 주: 여기서 “구조 기반 미래”는 구조 지속 진술과 구조 귀결형 미래를 함께 묶는 상위 범주이다.

- 구조 접근이 가능함
- 동일 구조가 유지된다는 전제 하에 결과가 구조적으로 이어짐
- 단, 이 범주는 두 가지로 나뉜다:
  - 구조 지속 진술
  - 구조 귀결형 미래

즉,  
“구조 기반 미래”는 하나의 단일 범주가 아니라  
미래를 다루되 여전히 구조 쪽에 속하는 두 하위 범주의 묶음이다.

### 3.1 구조 지속 진술

- 동일 구조가 유지되면 동일 패턴이 반복된다는 진술
- 개별 사건의 특정 시점 발생을 예측하는 것이 아님
- 구조의 연속성과 반복성을 기술
- 구조 접근이 가능하며, 그 반복성 자체가 이미 관측 가능한 경우

→ “이 구조에서는 이런 일이 계속 나타난다”

예:

- 책임 귀속 공백이 유지되면 사후 정합성 실패가 반복된다
- 병목 구조가 유지되면 같은 지점에서 문제가 다시 발생한다

이는 예측이라기보다  
구조의 지속성에 대한 진술이다.

### 3.2 구조 귀결형 미래

- 구조 자체가 결과를 사실상 닫고 있음
- 발생 여부는 확정적이며, 남아 있는 것은 시간 또는 표면화 시점뿐임
- 추가 모델링보다 구조 판정이 우선함
- 관측의 연장으로 간주 가능

→ “이 결과는 아직 드러나지 않았을 뿐, 구조상 닫혀 있다”

예:

- 누적 과부하 구조가 유지되면 결국 병목이 표면화된다
- 복구 불가능한 손상 조건이 충족되면 실패는 시간 문제만 남는다

이 경우의 미래 진술은  
예측이 아니라 구조 귀결의 판정에 가깝다.

---

## 4. 예측

- 미래 상태에 대한 가정
- 발생 여부 자체가 불확정
- 모델, 가정, 확률에 의존
- 관측 외 요소가 개입됨
- 구조 접근이 불완전하거나, 구조만으로 결과를 닫을 수 없음

→ “무엇이 일어날 수도 있다”

단, 예측 역시 단일한 한 덩어리가 아니다.  
구조가 일부 존재하더라도 예측이 되는 경우가 있다.

### 4.1 시간의존 예측

- 구조는 존재하고 일부 접근도 가능함
- 그러나 결과는 구조만으로 닫히지 않음
- 시간 경로 추적, 연속 상태 변화, 외란 여부, 임계점 도달 여부가 필요함
- 발생 여부 자체가 아직 열려 있음

→ “구조는 있지만, 실제 발생은 시간 속 상태 변화를 따라가야 한다”

예:

- 비가 올 가능성
- 특정 시점의 네트워크 혼잡 발생 여부
- 몇 시간 후 시스템 온도 임계치 도달 여부

중요한 점은  
이 범주는 구조가 없어서 예측이 되는 것이 아니라,  
**구조가 있어도 시간 경로를 따라가지 않으면 미래가 닫히지 않기 때문에 예측이 된다** 는 것이다.

즉,

- 비의 물리 조건은 구조 문제
- 실제 강수 발생 여부와 시점은 예측 문제

이다.

### 4.2 모델의존 예측

- 구조 접근이 불완전하거나 부분적임
- 모델, 통계, 확률, 외삽을 통해 미래 상태를 추정함
- 구조보다 모델의 가정이 더 큰 비중을 가짐

→ “구조를 직접 닫지 못하므로, 모델을 통해 추정한다”

### 4.3 외삽 / 추정

- 일부 관측값을 바탕으로 미관측 상태를 보정함
- 구조 접근이 충분하지 않아 간접 추정이 개입됨
- 관측과 예측의 중간처럼 보일 수 있으나, 엄밀히는 예측 계열에 속함

---

## 5. 구조 접근 관점에서의 분류

- **관측**  
  → 구조 접근 가능 상태  
  → 상태 확정  
  → 구조는 이미 닫혀 있음

- **구조 지속 진술**  
  → 구조 접근 가능 상태  
  → 반복 패턴은 열려 있으나 구조 자체는 설명 가능  
  → 개별 사건 예측이 아니라 구조 연속성 진술

- **구조 귀결형 미래**  
  → 구조 접근 가능 상태  
  → 발생 여부는 사실상 확정  
  → 시간만 미정

- **시간의존 예측**  
  → 구조 접근 가능 또는 부분 가능 상태  
  → 시간 경로 추적 없이는 확정 불가  
  → 상태 변화와 외란에 민감

- **예측 / 예상 / 예견**  
  → 구조 접근 불완전 상태  
  → 모델, 가정, 확률 의존

- **예지**  
  → 구조 접근 불가 상태

---

## 6. SIF와의 관계

SIF는 구조 접근이 불가능하거나 제한되는 상태를 의미한다.

이 상태에서 발생하는 전이는 다음과 같다:

- 관측 → 예측 계열로 전이
- 확정 상태 → 가정 기반 상태로 전환
- 구조 기반 판단 → 모델/추정 기반 판단으로 이동
- 구조 귀결 판정 → 시간의존 추정 또는 확률 모델로 후퇴

즉,

구조 접근이 차단될수록  
관측은 예측으로 대체되고,  
구조 귀결은 모델 기반 가정으로 대체된다.

---

## 7. 축 정리

### 확정성 축

- 관측 / 구조 귀결형 미래 → 확정
- 구조 지속 진술 → 개별 사건 확정이 아니라 패턴 지속 진술
- 시간의존 예측 / 예측 계열 → 비확정

### 시간 축

- 관측 → 과거 / 현재
- 구조 지속 진술 / 구조 귀결형 미래 / 예측 → 미래

즉, 시간만으로는 관측과 예측을 구분할 수 없다.

### 의존성 축

- 관측 → 구조
- 구조 지속 진술 → 구조
- 구조 귀결형 미래 → 구조
- 시간의존 예측 → 구조 + 시간 경로 추적
- 예측 → 모델 / 가정 / 확률

### 폐쇄성 축

- 관측 → 이미 닫힘
- 구조 귀결형 미래 → 구조만으로 닫힘
- 시간의존 예측 → 구조는 있으나 시간 경로 없이는 닫히지 않음
- 모델 기반 예측 → 구조만으로 닫히지 않으며 모델 의존
- 예지 → 닫힘 여부 자체를 구조적으로 검증할 수 없음

이 축이 중요하다.

왜냐하면  
**“구조 접근 가능”** 과  
**“구조만으로 결과가 닫힘”** 은  
같은 말이 아니기 때문이다.

---

## 8. 정리

관측은 확정된 상태를 기술하는 행위이며,  
예측은 비확정 상태를 가정하는 행위이다.

그러나 미래에 대한 진술이라고 해서 모두 예측은 아니다.

- 구조의 반복성을 말하는 것은 구조 지속 진술이다
- 구조 자체가 결과를 닫고 있는 경우는 구조 귀결형 미래이다
- 구조는 있으나 시간 경로 추적과 상태 변화 판단이 필요한 경우는 시간의존 예측이다
- 구조 접근이 불완전하고 모델/확률/가정에 의존하는 경우는 예측이다
- 구조적으로 설명 불가능한 결과 제시는 예지이다

따라서 중요한 기준은 시간이 아니라 다음이다.

- 구조에 접근 가능한가
- 상태가 확정되었는가
- 구조만으로 미래가 닫히는가
- 아니면 시간 경로 추적과 모델링이 필요한가

즉,

관측과 예측의 차이는 단순한 시간 차이가 아니라  
**구조 접근 가능성, 확정성, 폐쇄성의 차이** 이다.
