# Observation vs Prediction (관측 vs 예측)

## 개요

본 문서는 관측과 예측의 차이를  
시간이 아닌 **구조 접근 가능성** 의 관점에서 정리한다.

이 구분은  
SIF (Structural Inaccessibility Failure) 발생 시 나타나는  
인식 전이를 설명하기 위한 것이다.

---

## 1. 핵심 구분

- 관측: 확정된 상태의 기술  
- 예측: 비확정 상태에 대한 가정

이 차이는 시간이 아니라  
**상태의 확정성** 과 **구조 접근 가능성** 에서 발생한다.

---

## 2. 관측 (Observation)

- 이미 발생했거나 확정된 상태를 기술
- 구조의 존재와 관계를 기록
- 시간은 과거 또는 현재에 고정
- 불확실성 없음
- 외부 가정이나 모델 개입 없음

→ “무엇이 있는가”

---

## 3. 구조 기반 미래 (Structural Continuity)

- 동일 구조가 유지된다는 전제 하에 발생을 확정
- 발생 여부는 확정, 시간만 미정
- 관측의 연장으로 간주됨

→ “이 구조는 계속 발생한다 (시간 미정)”

※ 이는 예측이 아니라  
구조의 지속성에 대한 진술이다.

---

## 4. 예측 (Prediction)

- 미래 상태에 대한 가정
- 발생 여부 자체가 불확정
- 모델, 가정, 확률에 의존
- 관측 외 요소가 개입됨

→ “무엇이 일어날 수도 있다”

---

## 5. 구조 접근 관점에서의 분류

- 관측  
  → 구조 접근 가능 상태

- 구조 기반 미래  
  → 구조 접근 가능 상태 (시간만 미정)

- 예측 / 예상 / 예견  
  → 구조 접근 불완전 상태

- 예지  
  → 구조 접근 불가 상태

---

## 6. SIF와의 관계

SIF는 구조 접근이 불가능하거나 제한되는 상태를 의미한다.

이 상태에서 발생하는 전이는 다음과 같다:

- 관측 → 예측 계열로 전이
- 확정 상태 → 가정 기반 상태로 전환
- 구조 기반 판단 → 모델/추정 기반 판단으로 이동

즉,

구조 접근이 차단될수록  
관측은 예측으로 대체된다.

---

## 7. 축 정리

### 확정성 축

- 관측 / 구조 기반 미래 → 확정
- 예측 계열 → 비확정

### 시간 축

- 관측 → 과거 / 현재
- 구조 기반 미래 / 예측 → 미래

### 의존성 축

- 관측 → 구조
- 예측 → 모델 / 가정

---

## 8. 요약

관측은 확정된 상태를 기술하는 행위이며,  
예측은 비확정 상태를 가정하는 행위이다.

구조 기반 미래는 예측이 아니라  
관측의 시간적 확장이다.

이 구분은 시간의 문제가 아니라  
구조 접근 가능성의 문제이다.
