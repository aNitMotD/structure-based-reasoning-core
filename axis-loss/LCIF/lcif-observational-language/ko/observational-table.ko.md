# LCIF 최소 관측 문장 테이블

이 테이블은 LCIF 이후(Post-LCIF)  
사고가 고착되는 과정에서 반복적으로 등장하는  
**최소 인간 발화 신호**를 구조적으로 정리한 것이다.

본 테이블은:
- 개념을 정의하지 않으며,
- 책임을 귀속하지 않고,
- 해결책이나 처방을 제시하지 않는다.

---

## 최소 관측 문장 테이블

| 관측 지점 | 최소 관측 문장 | 구조적 판독 (내부 논리) |
|----------|----------------|--------------------------|
| 제약 증발 | “문서에는 있었는데 결과에는 없었다” | 상위 레이어에 정의된 제약 조건이 출력 레이어까지 상속·집행되지 못하고 전파 과정에서 유실됨. |
| 관찰자 부재 | “검수는 했지만 막히지는 않았다” | 제약 준수를 실시간으로 감시하는 관찰 레이어가 부재하여 위반 행위가 그대로 통과됨. |
| 주권 침해 | “자동화 이후 책임이 사라졌다” | 실행 주권이 자동화에 의해 잠식되며 책임 경계가 붕괴됨. |
| LCIF-B 발현 | “그럴듯한 설명만 남았다” | 제약 상속 실패를 인지하지 못한 상태에서 서술 정합성에 의해 오류가 합리화됨. |
| 패턴 확정 | “사람 실수로 보기엔 반복된다” | 개별 사례가 아닌 구조적 한계로 해석 가능한 반복 패턴이 관측됨. |
| 임계점 포기 | “하나하나 다 대조해볼 수는 없지 않나” | 관찰 비용이 임계치를 초과함에 따라, 제약 기준 검수를 포기하고 서술 기준으로 전이함. |
| 허위 안도 | “문제가 있었다면 읽을 때 티가 났을 거다” | 서술적 정합성을 관찰자 레이어의 정상 작동 신호로 오인하여 검수를 생략함. |

---

## 사용 위치 주의

이 테이블의 표현들은:
- 사고 진행 중 판단을 대체하는 언어로 사용되어서는 안 되며,
- 설득·합의·책임 설명의 도구로 사용되어서는 안 된다.

본 테이블은 **사후 판독과 구조 인식**을 위한  
관측 전용 기록이다.
