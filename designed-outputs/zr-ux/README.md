# ZR-UX

## Status
Archived.

---

## English

This directory exists to document a post-hoc design decision regarding ZR-UX.

ZR-UX was considered as a possible UX layer related to Zero Reflect (ZR).
After design validation, this approach was determined to be
structurally incompatible with ZR’s boundary-based design
and was formally closed.

This directory serves only as a record of that decision.
No implementation or operational artifacts are defined here.

---

## 한국어 (Korean)

이 디렉터리는 ZR-UX에 대해 내려진
사후 설계 판단을 기록하기 위해 존재합니다.

ZR-UX는 Zero Reflect(ZR)와 관련된 UX 레이어 개념으로 검토되었으나,
설계 검증 결과 ZR의 책임 경계 기반 설계와
구조적으로 양립할 수 없다고 판단되어
공식적으로 종료되었습니다.

본 디렉터리는 해당 판단을 기록으로 남기기 위한 용도이며,
구현물이나 운영 산출물은 정의되어 있지 않습니다.
