# Zero Reflect UX (ZR-UX)

ZR-UX는 Zero Reflect(ZR) 위에 얹힌 UX 레이어이다.

ZR-UX는 ZR의 권한이나 판단 범위를 확장하지 않는다.
ZR-UX의 역할은 판단을 제공하지 않으면서
책임 경계를 인식 가능하게 만드는 것이다.

## 범위
- UX 제약 하에서의 책임 경계 가시화
- 행동 유도 없는 최소한의 설명 허용
- 중단과 침묵을 유효한 UX 결과로 취급
- ZR 코어 제약에 종속된 UX 동작만 허용

## ZR과의 관계
- ZR 코어 제약이 항상 최우선이다.
- ZR-UX는 새로운 규칙이나 권한을 생성하지 않는다.
- UX 문서는 비규범적이며 언제든 제거 가능하다.

## 주의
- ZR-UX 사례는 “잘 설계된 UX”가 아니라
  “어디까지 말할 수 있었는지”를 기록한다.
- UX는 misuse 제약이나 코어 명세를 우회하지 않는다.

이 레포지토리는 **CC BY-NC-ND** 라이선스 하의  
사고엔진 관측 메모 및 구조 정리 문서이다.

본 레포지토리 내에서 언급되는 **ZR-UX (Zero Reflect UX)** 는  
이 레포의 하위 구성 요소가 아니며,  
별도의 **독립 레포지토리** 로 존재하는 시스템이다.

- ZR-UX는 **AGPL-3.0** 라이선스를 가진 제약·경계 시스템이다.
- 본 레포에 포함된 ZR-UX 관련 내용은
  구현물, 배포물, 실행 코드가 아니라
  **사후 정리·설명·관측 기록** 에 해당한다.

따라서 본 레포지토리는  
ZR-UX의 배포처, 실행 환경, 파생 대상이 아니다.
