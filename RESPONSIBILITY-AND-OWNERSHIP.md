# Responsibility and Ownership Statement

## English

This document clarifies responsibility boundaries
for all materials in this repository.

This document operates under
INTERPRETATION-GUARDRAILS.md.

All materials in this repository
preserve human responsibility by design.

No artifact in this repository:
- delegates judgment
- acts as a proxy authority
- assumes responsibility
  on behalf of users or organizations

Responsibility for interpretation,
decision-making, and application
always remains with the user or organization.

## Third-Party Intervention Boundary

This repository does not assume any structure in which responsibility  
is transferred to the author through third-party interpretation, claims,  
reports, or legal actions.

All actions and outcomes resulting from the interpretation or application  
of this repository remain the responsibility of the user or executing party,  
regardless of any third-party involvement.

# 책임 및 소유권 고지

## 한국어 (Korean)

본 문서는 본 레포지토리 내
모든 자료에 대한 책임 경계를 명확히 한다.

본 문서는
INTERPRETATION-GUARDRAILS.md
하에서 적용된다.

본 레포지토리의 모든 자료는
설계상 인간의 책임을 보존한다.

본 레포지토리의 어떠한 자료도:
- 판단을 대리하지 않으며
- 권위의 대체물이 되지 않으며
- 사용자 또는 조직을 대신하여
  책임을 부담하지 않는다.

해석, 판단, 적용에 대한 책임은
항상 사용자 또는 조직에 귀속된다.

## 제3자 개입에 대한 책임 경계

본 레포지토리의 내용은 제3자의 해석, 주장, 신고 또는 법적 행위에 의해  
작성자에게 책임이 전이되는 구조를 전제로 하지 않는다.

사용자가 본 레포지토리의 내용을 해석·적용하여 발생한 모든 행위와 결과는  
해당 사용자 또는 실행 주체의 책임에 속하며,  
제3자의 개입 여부와 무관하게 작성자에게 귀속되지 않는다.
