# Structural Boundary Analysis

## English

This directory contains **structural re-representations of existing RCA texts**.

Each sub-analysis decomposes an RCA into explicit dimensions such as:
- structural mechanism,
- responsibility boundary,
- failure path,
- and residual observations.

No new incidents, conclusions, or corrective actions are introduced here.  
This directory functions as a **view layer**, not as an independent case or output.

Interpretation, judgment, and responsibility remain with the original RCA text.

# 구조 경계 분석

## 한국어 (Korean)

이 디렉터리는 **기존 RCA 텍스트를 구조적으로 재표현한 분석** 을 포함한다.

각 분석은 하나의 RCA를 다음과 같은 축으로 분해한다:
- 구조적 메커니즘,
- 책임 경계,
- 실패 경로,
- 잔여 관측 요소.

본 디렉터리는 새로운 사건, 결론, 또는 조치를 추가하지 않는다.  
독립적인 케이스나 산출물이 아니라, **보조적 구조 뷰 레이어** 이다.

해석, 판단, 책임의 기준은 항상 원본 RCA 텍스트에 귀속된다.
