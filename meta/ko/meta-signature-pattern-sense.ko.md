# meta-signature-pattern-sense

## 1. 개념 정의

**Meta-Signature-Pattern-Sense (MSPS)** 는  
개별 감각의 강도·취향·정상성 이전에 존재하는  
**전처리 감지 레이어(pre-structural sensing layer)** 를 의미한다.

이 레이어는 감각 내용을 분류하지 않고,  
**판단에 필요한 신호가 언제·어디서 발생했는지** 만을 반환한다.

---

## 2. 최소 구조 좌표 (Core Axes)

MSPS는 다음 **4개 좌표 축** 으로 기술된다.

### A. 탐지 대상 (Detection Target)
- 연속 차이 (continuous variation)
- 이상치 / 위험 (outlier / hazard)
- 변화율 / 타이밍 (rate of change / timing)
- 정체성 서명 (identity signature)

### B. 작동 시점 (Activation Phase)
- 감정 이전 (pre-emotional)
- 판단 이전 (pre-judgment)
- 행동 직전 (pre-action)
- 사후 검증 (post-action)

### C. 목적 (Functional Intent)
- 쾌감
- 회피
- 분기 판단
- 책임 차단

### D. 출력 형태 (Output Signal)
- 미묘한 느낌
- 즉시 회피
- 멈칫 / 보류
- 확신 없는 경고

> 감각의 내용은 다르지만,  
> MSPS는 항상 이 4축 중 어딘가에 투영된다.

---

## 3. 작동 흐름

signature / pattern 감지  
→ 출력 신호 발생  
→ 구조화  
→ 판단  
→ 책임 귀속

MSPS는 **구조의 입력** 이지,  
구조 그 자체가 아니다.

---

## 4. 구조 센스와의 구분

### Signature / Pattern Sense (MSPS)
- 설명 이전
- 즉시 판별
- 이유 없는 “같다 / 다르다 / 위험하다”

### Structural Sense
- 관계·제약·경계 모델링
- 언어화 가능
- 사후 설명 가능

## 5. 개인 사례 (예시)

> 아래는 MSPS가 실제로 어떻게 투영될 수 있는지에 대한  
> **하나의 관측 사례**일 뿐, 일반 규칙이 아니다.

### 미각
- 탐지 대상: 연속 차이
- 목적: 조합 분해
- 출력: 미묘한 이질감

### 후각
- 탐지 대상: 이상치 / 위험
- 목적: 즉시 회피
- 출력: 강한 차단 신호

### 시각
- 탐지 대상: 변화율 / 타이밍
- 목적: 분기 판단
- 출력: 즉각 반응 가능 상태

### 청각
- 탐지 대상: 정체성 서명
- 목적: 초기 경로 판별
- 출력: “이건 그거다”라는 확신

### 촉각
- 탐지 대상: 접촉 경계의 불연속
- 목적: 원인 위치 고정
- 출력: 즉각적 인지 신호

> **후처리(Post-Processing)에 대한 주석**
>
> MSPS는 *해석 이전(pre-interpretive) 감지* 를 위한 최소 좌표계를 기술한다.
> 연속성, 지속성, 궤적과 같은 해석은 **MSPS 출력 이후의 후처리 레이어** 에 속하며,
> MSPS 축의 일부가 아니다.
>
> 연속형/이산형(스냅샷) 해석 스타일에 대한 논의는
> 아래의 별도 문서를 참조한다.
> - `msps-post-processing-notes.ko.md`

## 6. 결론

감각은 천차만별이지만,  
**감각이 판단 구조에 붙는 위치는 좌표화 가능** 하다.

MSPS는 감각을 분류하지 않는다.  
감각이 **언제, 무엇을 위해, 어떤 신호를 내는지** 만을 기술한다.

> **MSPS는 구조의 입력이며, 개인의 결합 형식과 데이터 특성의 매칭이 활성화를 결정하고,  
> 수준 판정은 비율이 아니라 전환 발생 여부로 한다.**
