# 일상 영역에서의 자격 요구 논리 반례 모음

> 목적:  
> 일상적 행위·관찰·사용에 대해 **전문 자격 검증을 요구하는 논리**가 어떻게 자기모순에 빠지는지를 보여주는 반례용 정리 문서.  
> 가상 서사에서 “자격 없는 자” 프레임을 반박하기 위한 **구조적 반례 카탈로그**.

---

## 1. 신체 사용

- 걷기 → 생체역학 전공 또는 물리치료사 자격 필요?
- 달리기 → 스포츠과학 학위 또는 마라톤 완주 기록 요구?
- 계단 오르기 → 하지 관절 역학 검증 통과?
- 넘어짐 경험 언급 → 안전공학 사고 분석 자격 필요?

**현실:**  
→ 인간은 자격 없이 이동한다.

---

## 2. 도구 사용

- 가위 사용 → 공구 취급 교육 이수?
- 칼 사용 → 조리사 자격 필수?
- 망치 사용 → 기능사 자격 요구?
- 드라이버 사용 → 전기·기계 복합 자격?

**현실:**  
→ 도구 사용은 전문가 선언이 아니다.

---

## 3. 전기·기술 인프라

- 220V 콘센트 사용 → 전기기술사 자격?
- 멀티탭 연결 → 전기안전관리자 인증?
- 스마트폰 충전 → 전력 시스템 이해도 검증?
- 와이파이 사용 → 네트워크 엔지니어 자격?

**현실:**  
→ 표준을 사용하는 것은 전문 행위가 아니다.

---

## 4. 이동 수단

- 자전거 타기 → 사이클 대회 출전 이력 요구?
- 자동차 운전 → 레이싱 대회 검증?
- 지하철 이용 → 철도공학 이해 시험?
- 비행기 탑승 → 항공역학 검증?

**현실:**  
→ 이용자는 설계자가 아니다.

---

## 5. 디지털 행위

- 엑셀 사용 → 데이터사이언스 학위 필요?
- 검색 사용 → 정보검색학 자격 요구?
- AI 사용 → 머신러닝 석사 이상?
- 코드 읽기 → 컴퓨터공학 전공 필수?

**현실:**  
→ 도구 사용은 학문 귀속을 요구하지 않는다.

---

## 6. 창작·표현

- 글 쓰기 → 문학상 수상 이력 필요?
- 그림 그리기 → 미대 졸업 증명?
- 음악 감상 후 평 → 작곡 전공 요구?
- 사진 촬영 → 포토그래퍼 인증?

**현실:**  
→ 표현은 허가 대상이 아니다.

---

## 7. 관찰·비판

- 위험하다고 말함 → 안전공학 자격?
- 구조가 이상하다고 지적 → 해당 학문 박사?
- 실패 지점 언급 → 연구 책임자 인증?
- 문제 제기 → 공인 검증 논문 선행 요구?

**현실:**  
→ 관측은 자격이 아니라 경험에서 발생한다.

---

## 핵심 정리 문장

> 이 논리를 끝까지 밀면,  
> **살아 있는 대부분의 행위는 자격증 없이는 말할 수 없는 행위가 된다.**

---

## 가상 서사용 반례 문장 예시

- “걷는 법을 알면 생체역학 검증부터 받아야 합니까?”
- “220V 콘센트를 쓰기 전에 전기기술사 시험부터 봐야 하나요?”
- “엑셀을 쓰려면 데이터사이언스 석사가 필요합니까?”
- “위험을 봤다고 말하는 데 자격증 번호가 필요합니까?”

---

## 구조적 결론

- 자격은 **책임을 위임받을 때** 필요하다.
- 관찰과 사용에는 자격이 필요하지 않다.
- 자격을 요구하는 순간, 논점은 내용이 아니라 **발언권 통제**로 이동한다.

---

자격 요구 논리의 자기모순을 드러내기 위한 반례 정리 자료이다.*
